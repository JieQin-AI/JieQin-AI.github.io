# JieQin-AI.github.io

Personal academic homepage hosted on GitHub Pages.

## Project Structure

```
├── index.html          # Main HTML page
├── images/             # Publication images and profile photo
├── .github/
```

This site is automatically deployed to GitHub Pages via GitHub Actions on every push to the `main` branch.
